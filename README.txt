Οι Αλλαγές που έκανα
	* Έφτιαξα xml configuration File στο φάκελο config 
	* Έκανα αυτόματη την παραγωγή csv οπότε κράτησα μόνο 1 κλάση για csv
	* Πρόσθεσα ένα φάκελο properties μέσα στο φάκελο Resources
	* Μέσα σε αυτό το φάκελο , υπάρχουν 3 αρχεία .txt, το ένα είναι για τα
	  metadata, το άλλο είναι για κάθε αρχείο csv να δηλώνεις ποια στήλη θα 
	  αποτελέσει μέρος του Subject κάθε γραμμής του excel και το τελευταίο είναι για
	  να αντιστοιχούμε κάθε στήλη με το property που θα βάλουμε.
	* Για το κατέβασμα από τη DBpedia μπορείς να δηλώσεις το Query στο configuration
	* Αν πατήσεις run θα δημιουργηθούν αυτόματα τα πάντα
Δες αν χρειάζονται αλλαγαές αλλιώς, αν είναι θα φτιάξω και τα javadoc και το στέλνουμε
